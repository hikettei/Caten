proceed is a function that evaluates a computational graph.

lisp
CATEN-USER> (proceed (make-tensor `(3 3) :initial-element 1.0))

ctx:with-contextvar is a macro that sets the context variable. BACKEND=CLANG to use JIT, JIT_DEBUG=4 to see the generated code. DOT=1 to debug the pattern matcher on your browser. (need graphviz)

lisp
CATEN-USER> (ctx:with-contextvar (:BACKEND "CLANG" :jit_debug 4 :dot 0)
  (caten (forward (Embedding 10 10) (make-tensor `(b c)))))
Result
